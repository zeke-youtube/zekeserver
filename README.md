# 🤖 ZekeBot

ZekeBot is a powerful, modular Discord bot designed for moderation, memes, utility commands, and more — all bundled into a clean slash-command experience.

> Built with love, curiosity, and the occasional meme 🧠⚡

---

## 🚀 Features

- `/mod` — Warn, ban, kick, unban, or clear messages  
- `/memes` — Fetches random Reddit memes via API  
- `/help` — Lists all available commands  
- `/source` — Shows this repo and bot code  
- Ephemeral replies (private, clutter-free responses)

---

## 📦 Getting Started

1. Clone the repo:
   ```bash
   git clone https://github.com/zeke-youtube/zekeserver.git
   ```
2. Create a `token.txt` file and paste your bot token inside it.
3. Run the bot:
   ```bash
   python bot.py
   ```

---
